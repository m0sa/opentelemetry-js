Class OpenCensusMetricProducer

A MetricProducer which collects metrics from OpenCensus. Provide an instance to your MetricReader when you create it to include all OpenCensus metrics in the collection result:

Example

const reader = new PeriodicExportingMetricReader({
  metricProducers: [new OpenCensusMetricProducer()],
  exporter: exporter,
});
const meterProvider = new MeterProvider({
  readers: [reader],
});
